About:
  This is a typing speed test. The user is shown a text or a set of words that he must type correctly.

Parameters:				Values:					        Description:

  <level>               "easy" / "medium" / "hard"                          Game difficulty level.
  <time_limit>          5-180						    Time limit.
  <max_errors>          1-30						    Maximum number of errors.

Restart the program.
Enjoy!
